# Blink

A discreet, secure file-sharing app for two users.

## What it does

Blink lets two people share photos, videos, and documents privately. Shared content is never saved to the device or the cloud — it exists only inside the app. If the receiver wants to keep something, they must request permission. The sender gets a yes/no prompt and decides whether to allow it.

## Core principles

- **No local storage** — received files are not written to the device gallery or filesystem
- **No cloud storage** — nothing is stored on a server after delivery
- **Sender controls saves** — the receiver cannot save content without explicit approval from the sender
- **End-to-end encrypted** — content is encrypted in transit between the two users
